# MemoApp
MemoApp es una aplicación que te ayuda a no olvidarte de lo importante. Posee las siguientes funciones:
* Login/Registro de cuenta.
* Notas: podés crear tus propias notas para anotarlo todo.
* Lista de compras: te permite realizar una lista de las cosas a comprar en la semana para no olvidarte de comprar nada.
* Pastillero: te permite agendar los medicamentos que tenés que tomar de forma semanal o eventual. Te avisa por medio de una alarma a la hora de tomarlo.
* Turnos médicos: tiene su propio calendario para que agendes los turnos médicos. Te avisa con una alarma antes del turno para que lo recuerdes.
* Cumpleaños: agendá los cumpleaños de tus seres queridos, y no te olvidés nunca más de saludarlos.
* Binoculares: ¿te olvidaste los lentes? No hay problema, sacale una foto al papel así la app te muestra el texto en pantalla.
* SOS: ¿te perdiste o necesitás ayuda? apretá el botón de sos que enviará un mensaje automático a un contacto elegido, con tu ubicación.
